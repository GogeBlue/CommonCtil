◆ cmtSpinLockEnter()

BOOL CMT_FASTCALL cmtSpinLockEnter ( cmtUint8 *  value,
cmtUint64  MaxSpin 
)

进入自旋锁

参数
[in]value锁变量地址
[in]MaxSpin最大自旋数（-1为不限）
返回
返回原因
返回值
0锁被释放
1自旋数到达上限
注意
value为0时才允许进入自旋锁，所以第一次调用前请将value归零
示例:
cmtUint8 lock = 0;
cmtSpinLockEnter(&lock, -1);
//xxxxxx
cmtSpinLockLeave(lock);
